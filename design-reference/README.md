# MoneyTrack UI Reference

请把两张 UI 参考图保存到本文件夹，并使用以下文件名：

- `moneytrack-ui-reference-1.png`
- `moneytrack-ui-reference-2.png`

当前实现应参考这两张图的移动端方向：

- Android 本地 APK App
- 底部 Tab 导航
- 白色背景、浅灰页面底色
- 蓝色主强调色
- 圆角卡片
- 首页银行 App / 记账 App 风格
- 交易记录使用手机卡片列表
- 支出红色、收入绿色、普通信息蓝色或灰色

终端无法直接访问本对话中图片附件的原始文件，所以需要先把图片文件放到本目录。
